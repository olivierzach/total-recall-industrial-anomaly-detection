# 2026-03-11

## PatchCore / industrial anomaly detection

### What changed
- Added PRO metric implementation + integration into BTAD eval output.
- Added fast-iteration doc (`docs/DEV_CONFIG.md`).
- Added Apple Silicon GPU (MPS) support notes + benchmark and speed comparison tooling.
- Added a "from scratch" doc for running on a new machine + fitting on your own nominal images.
- Added a learned reference selection / supervised coreset plan doc.
- Added BTAD sweep tooling:
  - exposed additional eval knobs: `--num-neighbors`, `--image-score`, `--(no-)l2-normalize`
  - added `scripts/sweep_btad.py` to run a grid of configs and summarize results

### Key results observed
- Full BTAD ViT-B/16 @ coreset_ratio=0.0005: image AUROC ~0.8285, pixel AUROC ~0.7427, PRO~0.1914.
- Smoke benchmark shows MPS speeds up feature extraction substantially (3–4x), but end-to-end speedup is smaller due to CPU-bound metric computation.
